# Szolnoki Fitness Terem Weboldal

Ez a projekt a **Szolnoki Fitness Terem** bemutató weboldala, amely React és Bootstrap 5 alapokon készült.  
Tartalmaz **hero szekciót, szolgáltatások listáját, árlistát, kapcsolat űrlapot és footer-t**.

---

## Főbb funkciók

- Reszponzív hero szekció háttérképpel
- Kártyák a szolgáltatások és árak bemutatására (map-elés a komponensekben)
- Kapcsolat űrlap valós idejű validációval
- Hibajelzés és sikeres elküldés üzenet
- Footer linkek közösségi média gombokkal (új lapon nyitás)
- Bootstrap 5 komponensek a könnyű stílusért és reszponzivitásért

---

## Telepítés

1. Klónozd a repository-t:
   git clone <repository-url>

2. Lépj be a projekt könyvtárba:
   cd <project-folder>

3. Telepítsd a függőségeket:
   npm install

4. Indítsd el a fejlesztői szervert:
   npm start
